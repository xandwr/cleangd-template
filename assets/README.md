# assets

Use this for imported media and source files: models, sprites, textures, audio,
fonts, materials, shaders, VFX textures, and art source files. This folder is
for content, not behavior.

Good examples:

- `assets/3d/characters/player/player.glb`
- `assets/3d/props/door/door.glb`
- `assets/2d/icons/interact.png`
- `assets/audio/sfx/ui_confirm.wav`
- `assets/audio/music/main_theme.ogg`
- `assets/fonts/interface/Inter.ttf`
- `assets/materials/metal_brushed.tres`
- `assets/shaders/wind_grass.gdshader`
- `assets/source/blender/player.blend`

Avoid putting gameplay scenes here unless they are pure visual asset scenes.
Interactive prefabs usually belong in `game/`.
